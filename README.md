# umass-cics250-regex-game.github.io

This repository contains the source code for a Regular Expressions game, intended for use in the UMass Amherst CICS course COMPSCI 250. This project was created by Jackson Bibbens, Carter Luck, Ayesha Binte Mostofa, and Sribatscha Maharana as a course project for COMPSCI 879 in Fall 2025.

## üéØ Purpose & Learning Goals

This repository hosts an **interactive educational web game** designed to help students learn and practice regular expressions (regex) in a structured and engaging way. Through pre- and post-quizzes and a hands-on Password Game, students can:

- Apply regex concepts to solve practical string-matching challenges.
- Reinforce understanding of patterns, repetitions, sets, and advanced regex constructs.
- Track and improve skill level through pre- and post-assessments.
- Encourage hands-on learning in a gamified, interactive environment.
