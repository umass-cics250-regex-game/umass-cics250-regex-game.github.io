CS 250 Regular Expression Game

Welcome!

This site is intended to help CS 250 students practice identifying and using regular expressions. Understanding regular expressions can be helpful not only for this course, but also has connections to applications of pattern matching seen in programming languages, compilers, and search engines. Through this activity, students will gain the following skills

  • Identify when a string matches a regular expression
  • Create regular expressions which follow a given set of strings
  • Discover how small pattern changes affect string behavior

In order to measure the effectiveness of this activity, you will complete a short knowledge check before and after the game and your scores will be recorded. These scores are completely anonymous and are not part of your course grade.

Regular Expression Game

At each level of the game, you will input a regular expression which includes or excludes the given strings. Your solution should only include the following characters { 1 , 0 , ( , ) , * , + }

You must pass at least 10 levels to continue to the Post-Game Knowledge Check, but you are welcome to play for longer.
